# myofibre-cross-sectional-area
Data filtering and visualisation workflow for immunohistochemistry derived myofibre cross-sectional area analyses

# Why This Project
Measuring skeletal muscle cross-sectional area directly through immunohistochemistry is imperative for understanding how exercise training and/or nutritional interventions alter myofibre composition. As a researcher with a background in sport and exercise science, I built this project to demonstrate how large immunohistochemistry datasets can be easily filtered by cutoff criteria and visualised with an automated script for researchers to draw insights from their data with ease.

# Tools
R · tidyverse · ggplot2 · janitor · nlme · lsmeans · patchwork

# Data Source
B. Shamim (unpublished data, 2026)

# How to Run
Clone this repo
Open in RStudio
Run scripts/fibre_csa_analysis.R
